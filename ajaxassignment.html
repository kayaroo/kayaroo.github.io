Explanation of AJAX

Ajax stands for Asynchronous JavaScript And XML. Ajax is a technique used for communicating with a server without the need to refresh. For example in this case, " cd_catalog.xml" is used for sending all requests onto the file "ajaxassignment.html". You're able to code from the former while the ladder changes there without the need of having to refresh the webpage.

REST API

The API that I chosen is Cat Facts API. What this does is that it it allows you to send text messages on random facts about cats daily. They include multiple features. For example, they include a timer which keeps track on the countdown on when the next set of cat facts appears. They even allow you to send your own facts that you know about cats as well. It uses response files like JSON, JS, YML, MD, gitignore, and bowerrc. It includes a Catbot that does multiple things like texting back when you text and adding recipients and allows you to view the conversation. It can even be able to import all the people you have in your google contacts in one go.
